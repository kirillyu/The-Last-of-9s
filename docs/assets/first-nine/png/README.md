# Translation PNG assets (First Nine)

This folder contains **lossless PNG** copies of the `first-nine` diagrams, renamed to be clearly tied to the article and the image order inside it.

- `ru/` contains the actual PNG files.
- `en/` currently contains symlinks to `ru/` (until English screenshots replace them).
- `mapping.json` maps original hashed `.webp` filenames to the new PNG names.

Naming: `<article-markdown-stem>_<NN>.png` (example: `2025-12-21-runtime-models_03.png`).
