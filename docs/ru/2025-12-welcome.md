# Пример статьи - Добро пожаловать

Это примерная статья для демонстрации работы вашего блога.

## Структура статьи

Каждая статья должна содержать:

1. **Front matter** в YAML формате с полями:
   - `title` - заголовок статьи
   - `date` - дата публикации (YYYY-MM-DD)
   - `description` - краткое описание
   - `tags` - список тегов

2. **Содержимое** в Markdown формате

## Пример кода

```go
package main

import "fmt"

func main() {
    fmt.Println("Hello, Blog!")
}
```

## Следующие шаги

Вы можете:
- Удалить эту статью
- Добавить свои статьи в `docs/ru/` или `docs/en/`
- Запустить `make serve` для локального просмотра
- Закоммитить изменения для автоматической публикации
