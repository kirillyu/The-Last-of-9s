# Дашборды

Grafana дашборды сгруппированы по сигналам и слоям системы. Здесь будут live панели, статические
снимки и короткие runbook.

<div class="dash-grid">
  <div class="dash-card">
    <h3>Latency Terrain</h3>
    <p>Перцентили, насыщение, рост хвоста и распространение латентности.</p>
    <p class="post-meta">Статус: в сборке</p>
  </div>
  <div class="dash-card">
    <h3>Reliability Radar</h3>
    <p>SLO heatmap, error budget, влияние релизов.</p>
    <p class="post-meta">Статус: в сборке</p>
  </div>
  <div class="dash-card">
    <h3>Capacity Forge</h3>
    <p>CPU, память, IO, очереди, запасы для масштабирования.</p>
    <p class="post-meta">Статус: в сборке</p>
  </div>
  <div class="dash-card">
    <h3>Incident Lens</h3>
    <p>Связь трасс, логов и ошибок вокруг инцидентов.</p>
    <p class="post-meta">Статус: в сборке</p>
  </div>
</div>

## Примечания

- Часть дашбордов будет доступна в live виде.
- Для закрытых дашбордов будет статичный снапшот с аннотациями.
- К каждому дашборду будет короткий runbook.
